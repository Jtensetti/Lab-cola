# Mixture B — Water-based solution (1 L)

Into ~200 mL **hot** water, add sequentially (fully dissolve/mix each before adding the next):

| Ingredient | Amount |
|---|---:|
| Vinegar (5%) | 10 mL |
| Caffeine | 9.65 g |
| Glycerin | 175 g |
| Phosphoric acid (85%) | 45 mL |
| Wine tannins | 8 g |
| Vanilla extract | 10 mL |
| Caramel color (e.g., Shank’s) | 320 mL |

Then dilute with water to a **final volume of 1 L**.
