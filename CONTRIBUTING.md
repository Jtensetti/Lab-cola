# Contributing

Pull requests are welcome.

Please:
- Keep the project **documentation-first**.
- Avoid copying copyrighted tables/figures from paywalled papers.
- Prefer citing DOIs and publicly accessible abstracts/metadata.
- If you add variants (e.g., different caramel colors, acids, sweeteners), document:
  1) what changed
  2) why
  3) how it affected flavor
  4) any safety implications
